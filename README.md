# Rivals Pack Assets

Файлы замены (звуки, текстуры, меши, шрифты) для косметических паков Roblox Rivals через [Fleasion](https://github.com/fleasion/Fleasion).

Структура плоская: все файлы в корне, на них ссылаются `.json`-конфиги Fleasion по raw-ссылкам вида:

```
https://raw.githubusercontent.com/SlavaLitus-hash/rivals-pack-assets/main/<файл>
```

Только косметика. Все ассеты — из свободных источников или сделаны нами.
