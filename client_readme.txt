MARVEL x NARUTO PACK for Roblox Rivals (Fleasion)
===================================================

HOW TO INSTALL
--------------
1. Open Fleasion Dashboard.
2. Import the config file: Jay.json
   (Profiles / Import -> select Jay.json)
3. Make sure the imported profile is ENABLED and any older
   profiles are disabled or removed.
4. IMPORTANT — before the first launch: clear the Roblox cache
   (Fleasion tray menu -> Clear Cache, or just keep the default
   "Clear Cache on Launch" option enabled). Roblox must re-download
   assets through the proxy, otherwise you will hear/see originals.
5. Launch Roblox through Fleasion and join Rivals.

WHAT THIS PACK CHANGES
----------------------
* Sounds (60+ replacements):
  - kills 1/2/3, headshot, head & body hit sounds
  - round/match win & lose themes, game result, match point
  - duel timers (taiko drums), queue countdown
  - lobby music (custom track, seamless loop + muffled version)
  - intro / load-in theme that flows into the lobby music
  - UI: clicks, map select, matchmaking, reward, rank up/down
  - weapons: sniper, crossbow, RPG, molotov, fists, katana
  - jump pads, portals, dash & equipment whooshes, battle horn
* Rank icons: 16 custom kunai emblems (bronze -> onyx) with
  shuriken sub-levels and tier-colored glow
* Map preview images (31 maps)
* Skyboxes: Station, Dimension, Graveyard, Graveyard Sudden Death
  and the shared daytime sky group
* UI icons (27): logo, HP, damage indicator, buttons, menu icons
* Item slot backgrounds (5)
* Ambience removal tweaks

KNOWN LIMITATIONS
-----------------
* After Rivals game updates, single replacements may stop working —
  the developers sometimes change asset IDs. This is normal and
  fixable: the pack just needs a small ID update, not a rebuild.
* The medkit sounds are intentionally NOT modified.
* Maps Battle Grounds / Default / Docks / Iceberg / Village share
  one physical skybox in the game files, so they all use the same
  custom sky - this is a game limitation, not a bug.

Enjoy!
