========================================================
  x6ph — STANDARD PACK for Rivals (Fleasion)   v1.0
========================================================

Thanks for your order! This is a focused Standard pack — it changes only
the items listed below, nothing else in the game is touched.

--------------------------------------------------------
WHAT THIS PACK CHANGES
--------------------------------------------------------
  • Head Hit sound  — your headshot hitmarker (two in-game variants,
                      second one very slightly higher pitched).
  • Body Hit sound  — your body hitmarker. Mixed HOT on purpose:
                      noticeably louder / punchier than the default,
                      peaks kept just under 0 dB so it never clips.
  • Logo            — the Rivals logo image.
  • Arena texture   — the arena surface texture.
  • Skybox          — ALL maps get the Purple Nebula sky.
                      (Station has its own set of faces; every other
                      map shares one set. On Graveyard / Graveyard SD /
                      Zombie Tower / Westown the game uses a single
                      texture for all four side walls, so those four
                      walls all show the same nebula face — expected.)

--------------------------------------------------------
HOW TO IMPORT (Fleasion)
--------------------------------------------------------
  1. Open Fleasion.
  2. Import / Add config  ->  select the file:
        x6ph_Standard_V1.0.json
  3. Make sure the config is ENABLED (toggle on).
  4. All assets stream from the cloud (CDN) — no extra files to place,
     you don't need anything except the .json.

--------------------------------------------------------
IMPORTANT: CLEAR THE ROBLOX CACHE BEFORE TESTING
--------------------------------------------------------
  Roblox caches assets on disk. If you skip this you may still see/hear
  the OLD assets and think the pack didn't work.

  1. Fully CLOSE Roblox (quit completely, not just the window).
  2. In Fleasion, remove any older/previous imports of this pack.
  3. Enable "Clear Cache on Launch" in Fleasion (or clear the Roblox
     cache manually), then launch.
  4. Join a match and check hit sounds / skybox / arena.

  "Nothing changed" almost always means a stale cache — clear it and
  relaunch before reporting an issue.

--------------------------------------------------------
HEADS-UP: GAME UPDATES
--------------------------------------------------------
  Rivals sometimes changes the internal IDs of its assets when the game
  updates. If that happens, one or two things in this pack may stop
  applying (usually the skybox or a hit sound). This is NOT a broken
  pack — the IDs just moved. Message me and I'll patch the affected
  IDs quickly.
